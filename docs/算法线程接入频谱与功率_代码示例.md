# 算法线程接入频谱与功率（代码示例）

本文基于你当前三线程主链（采集线程 A -> 算法线程 B -> 日志线程 C）给出最小侵入示例：

- 在算法线程中对预处理后的单通道 `y` 做 Welch PSD。
- 计算 Delta/Theta/Alpha/Beta/Gamma 功率与 `alpha/beta` 比值。
- 保留现有主 CSV（`raw+preproc` 一行合并）不变。
- 新增一个频谱功率 CSV（窗口级一行），避免主 CSV 体积爆炸。

---

## 1) 设计建议

### 为什么不要把完整频谱点写入主 CSV

你的主 CSV 是逐样本（sample-level），而频谱是逐窗口（window-level）：

- sample-level：`tsMs, seq, rawInt16, signalQuality, rawUv, preprocUv`
- window-level：`seqStart, seqEnd, fs_used, delta, theta, alpha...`

两者时间粒度不同，强行合并会导致列语义混乱和文件过大。

建议：

1. 主 CSV 保持现状（兼容你已有逻辑）。
2. 新增 `spectrum_*.csv` 保存窗口级特征。

## 3) 把 PSD 计算独立为单独类（你要求的方案）

你希望“PSD 不要和算法线程逻辑耦合成黑盒”，建议新建：

- `preprocessing/psd_feature_extractor.h`（或放在 `core/psdfeatureextractor.h`，与 `CMakeLists.txt` 中路径一致即可）
- `preprocessing/psd_feature_extractor.cpp`

该类职责单一：输入 `y + fs`，输出频段功率，不依赖 UI 与线程对象。

### 3.0 编译错误说明（你遇到的 MSVC / MinGW 问题）

1. **`No template named 'QVector'`**  
   头文件里使用 `QVector` 必须包含 Qt 容器头。推荐至少：

   ```cpp
   #include <QVector>
   ```

   若工程里 `QVector` 仍不可见，再确认 `CMakeLists.txt` 已 `target_link_libraries(... Qt::Core ...)`，且该头参与编译的翻译单元能解析 Qt 包含路径。

2. **`Default member initializer for 'minSamples' needed within definition of enclosing class 'PsdFeatureExtractor'`**  
   这是把带**默认成员初始化**的 `struct Options` **嵌套在类 `PsdFeatureExtractor` 内部**时，部分编译器（含你用的 MinGW/Clang 前端）会报错：外层类尚未完整，`Options` 的成员默认值解析顺序不满足要求。  
   **解决办法（推荐）**：把 `Options` 挪到**命名空间作用域**（与 `EegPreprocessPipeline` 拆 `Options` 的做法一致），或去掉嵌套 `Options` 内所有 `= 默认值` 改在 `.cpp` 里手写默认构造。

下面示例采用「**类外 `PsdFeatureExtractorOptions` + 双构造函数**」，与 `EegPreprocessPipeline` 同款，可避免上述错误。

### 3.1 头文件示例：`psd_feature_extractor.h`

```cpp
#pragma once

#include <QVector>

struct PsdFeatureResult
{
    bool ok = false;
    int psdLen = 0;
    double delta = 0.0;
    double theta = 0.0;
    double alpha = 0.0;
    double beta = 0.0;
    double gamma = 0.0;
    double alphaBetaRatio = 0.0;
};

// 必须放在类外：避免嵌套 struct + 默认成员初始化器触发编译器限制
struct PsdFeatureExtractorOptions
{
    int minSamples = 32;
    int minNfft = 32;
    int maxNfft = 512;
    int window = 1; // WindowOperations::HANNING
    double eps = 1e-12;

    double fDeltaLo = 1.0,  fDeltaHi = 4.0;
    double fThetaLo = 4.0,  fThetaHi = 8.0;
    double fAlphaLo = 8.0,  fAlphaHi = 13.0;
    double fBetaLo  = 13.0, fBetaHi  = 30.0;
    double fGammaLo = 30.0, fGammaHi = 45.0;
};

class PsdFeatureExtractor
{
public:
    using Options = PsdFeatureExtractorOptions;

    PsdFeatureExtractor();
    explicit PsdFeatureExtractor(const Options &opt);

    void setOptions(const Options &opt);
    const Options &options() const;

    PsdFeatureResult compute(const QVector<double> &y, int fs) const;

private:
    Options m_opt;
};
```

### 3.2 源文件示例：`psd_feature_extractor.cpp`

```cpp
#include "psd_feature_extractor.h"
#include "data_filter.h"
#include "brainflow_constants.h"

#include <algorithm>
#include <vector>

PsdFeatureExtractor::PsdFeatureExtractor()
    : PsdFeatureExtractor(Options())
{
}

PsdFeatureExtractor::PsdFeatureExtractor(const Options &opt)
    : m_opt(opt)
{
}

void PsdFeatureExtractor::setOptions(const Options &opt) { m_opt = opt; }
const PsdFeatureExtractor::Options &PsdFeatureExtractor::options() const { return m_opt; }

PsdFeatureResult PsdFeatureExtractor::compute(const QVector<double> &y, int fs) const
{
    PsdFeatureResult out;
    if (fs <= 0 || y.size() < m_opt.minSamples)
        return out;

    std::vector<double> sig(y.begin(), y.end());

    int nfft = DataFilter::get_nearest_power_of_two((int)sig.size());
    if (nfft > (int)sig.size()) nfft /= 2;
    nfft = std::min(nfft, m_opt.maxNfft);
    if (nfft < m_opt.minNfft) return out;

    const int overlap = nfft / 2;
    int psdLen = 0;
    auto psd = DataFilter::get_psd_welch(
        sig.data(),
        (int)sig.size(),
        nfft,
        overlap,
        fs,
        m_opt.window,
        &psdLen);

    out.psdLen = psdLen;
    out.delta = DataFilter::get_band_power(psd, psdLen, m_opt.fDeltaLo, m_opt.fDeltaHi);
    out.theta = DataFilter::get_band_power(psd, psdLen, m_opt.fThetaLo, m_opt.fThetaHi);
    out.alpha = DataFilter::get_band_power(psd, psdLen, m_opt.fAlphaLo, m_opt.fAlphaHi);
    out.beta  = DataFilter::get_band_power(psd, psdLen, m_opt.fBetaLo,  m_opt.fBetaHi);
    out.gamma = DataFilter::get_band_power(psd, psdLen, m_opt.fGammaLo, m_opt.fGammaHi);
    out.alphaBetaRatio = (out.beta > m_opt.eps) ? (out.alpha / out.beta) : 0.0;
    out.ok = true;

    delete[] psd.first;   // ampl
    delete[] psd.second;  // freq
    return out;
}
```

### 5.2 `csvlogworker.cpp` 新增实现（示例）

```cpp
void CsvLogWorker::writeSpectrumHeaderIfNeeded()
{
    if (!m_spectrumFile.isOpen() || m_spectrumHeaderWritten)
        return;
    if (m_spectrumFile.size() == 0)
    {
        m_spectrumOut
            << "tsMs,seqStart,seqEnd,fsUsed,rms,delta,theta,alpha,beta,gamma,alphaBetaRatio,label\n";
        m_spectrumOut.flush();
    }
    m_spectrumHeaderWritten = true;
}

void CsvLogWorker::writeSpectrumRow(const SpectrumResult &sp)
{
    m_spectrumOut
        << sp.tsMs << ','
        << sp.seqStart << ','
        << sp.seqEnd << ','
        << QString::number(sp.fsUsed, 'f', 3) << ','
        << QString::number(sp.rms, 'f', 6) << ','
        << QString::number(sp.delta, 'f', 6) << ','
        << QString::number(sp.theta, 'f', 6) << ','
        << QString::number(sp.alpha, 'f', 6) << ','
        << QString::number(sp.beta, 'f', 6) << ','
        << QString::number(sp.gamma, 'f', 6) << ','
        << QString::number(sp.alphaBetaRatio, 'f', 6) << ','
        << sp.label
        << '\n';
}

void CsvLogWorker::onSpectrumResult(const SpectrumResult &sp)
{
    if (QThread::currentThread() != thread())
        return;

    if (!m_spectrumFile.isOpen())
    {
        if (m_spectrumCsvPath.isEmpty())
            return;
        m_spectrumFile.setFileName(m_spectrumCsvPath);
        if (!m_spectrumFile.open(QIODevice::WriteOnly | QIODevice::Append | QIODevice::Text))
            return;
        m_spectrumOut.setDevice(&m_spectrumFile);
        m_spectrumOut.setEncoding(QStringConverter::Utf8);
        m_spectrumHeaderWritten = false;
    }

    writeSpectrumHeaderIfNeeded();
    writeSpectrumRow(sp);
    m_spectrumOut.flush();
    m_spectrumFile.flush();
}
```

## 7) 文件命名建议

保持你现有目录 `QtBCI_logs`，新增频谱文件，不新增目录：

- `eeg_yyyyMMdd_HHmmss.csv`（主 CSV，逐样本）
- `spectrum_yyyyMMdd_HHmmss.csv`（频谱功率，逐窗口）
- `ui_yyyyMMdd_HHmmss.txt`（UI 操作日志）

---

## 8) 注意事项

1. `DataFilter::get_psd_welch` 返回的 `ampl/freq` 必须 `delete[]`。  
2. `nfft` 建议为 2 的幂，且 `nfft <= y.size()`。  
3. 频段功率计算前确保 `fs_used` 合理（在线估算失败时回退 nominal）。  
4. 不建议把完整 PSD 频点写主 CSV，会显著增加 I/O 压力。  
5. 单通道设备下 band power 更适合作趋势特征，不应过度解释绝对数值。

void CsvLogWorker::onSpectrumResult(const SpectrumResult &sp)
{
    if (QThread::currentThread() != thread())
        return;

    // 关闭 PSD 开关时，主动关闭文件句柄
    if (m_spectrumCsvPath.isEmpty())
    {
        if (m_spectrumFile.isOpen())
        {
            m_spectrumOut.flush();
            m_spectrumFile.flush();
            m_spectrumFile.close();
        }
        m_spectrumHeaderWritten = false;
        return;
    }

    if (!m_spectrumFile.isOpen())
    {
        m_spectrumFile.setFileName(m_spectrumCsvPath);
        if (!m_spectrumFile.open(QIODevice::WriteOnly | QIODevice::Append | QIODevice::Text))
        {
            emit workerError(QStringLiteral("Spectrum CSV open failed: %1")
                                 .arg(m_spectrumFile.errorString()));
            return;
        }
        m_spectrumOut.setDevice(&m_spectrumFile);
        m_spectrumOut.setEncoding(QStringConverter::Utf8);
        m_spectrumHeaderWritten = false;
    }

    writeSpectrumHeaderIfNeeded();
    writeSpectrumRow(sp);
    m_spectrumOut.flush();
    m_spectrumFile.flush();
}
